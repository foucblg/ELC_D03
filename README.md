# ELC-D03

Projet de l'électif "Applications Web" : recréer un [r/place](https://en.wikipedia.org/wiki/R/place)

## Configurer le projet

Dans `backend`, dupliquer `.env.template` en `.env`, et mettre les bonnes valeurs.

## Lancer le projet

### Windows

```sh
node .\back\server.js
```

### Autre (Unix-based)

```sh
node back/server.js
```
